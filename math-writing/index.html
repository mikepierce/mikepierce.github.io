A Template Webpage using math-writing.css

Mike Pierce

Abstract

This page has two purposes: to serve as a general tutorial on how one can publish mathematical content on webpages, and also to showcase math-writing.css. math-writing.css provides CSS classes that emulate some of nice theorem-like environments you can create in LaTeX using the amsthm package. This, in conjunction with MathJax, makes writing mathematics on a website straightforward. My hope is that providing this will encourage mathematicians to stop using WordPress blogs that are clunky and display mathematics poorly, and to stop typing up useful expository writing as a PDF, which is harder for search engines to index and notably harder to make accessible than content on a webpage.
